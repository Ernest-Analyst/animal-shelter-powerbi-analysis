Animal Shelter Data Analysis (Power BI)

📊 Overview

This project analyzes animal shelter operations to uncover insights into intake trends, outcomes, and operational efficiency.

🎯 Objectives

1. Identify high-volume intake periods
2. Analyze adoption and euthanasia rates
3. Measure average length of stay
4. Detect patterns in repeat intakes

🛠 Tools Used

1. Power BI
2. DAX
3. Data Modeling

📈 Key Insights

1. Adoption rates vary significantly by intake type
2. Certain animals experience longer stays
3. Seasonal intake trends impact shelter capacity
